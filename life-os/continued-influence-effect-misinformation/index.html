When Dealing with Misinformation: - Fact-Check: Verify the Source of Any Information Before Accepting

Combat the Continued Influence Effect

When dealing with misinformation: - Fact-check: Verify the source of any information before accepting it. - Mentally replace: Once corrected, actively focus on the updated information. - Repeat the truth: Reinforce the correct version to overwrite the false one. Example: If a rumor about a coworker turns out false, remind yourself of the actual facts whenever you think about them.
